# Student_Feedback_System
The Student Feedback System is a web-based application developed using PHP and MySQL to allow students to submit feedback online. Feedback is stored securely in a MySQL database and displayed in a clean, tabular format.

This project was completed as part of a Global Class collaboration between Universiti Malaysia Sarawak (UNIMAS) and Khushal Khan Khattak University Karak (KKKUK). The task was to implement a fully functional feedback system accessible worldwide.

Features

Online feedback submission form

Input validation for name and message

Real-time storage in MySQL database

Display of all feedback in a responsive table

Developer credit included

Technologies Used

PHP – Server-side scripting

MySQL – Database management

HTML & CSS – Front-end interface

Installation

Clone or download the repository:

git clone https://github.com/zohaib-zohi/Student_Feedback_System.git


Upload the files to a PHP-enabled server (e.g., InfinityFree, XAMPP).

Update db_connect.php with your database credentials:

$host = "YOUR_HOST";
$user = "YOUR_DB_USER";
$pass = "YOUR_DB_PASSWORD";
$db   = "YOUR_DB_NAME";


Access index.php in your browser.
